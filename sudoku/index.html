How to Play

Complete the board so each row, column and 3×3 box contains each number from 1 through 9 exactly once.

  • Select an empty square and enter a number.
  • Use Notes for small candidate numbers.
  • Use Check when you want feedback.
  • Use Hint if you get stuck.
