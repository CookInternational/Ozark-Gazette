# The Ozark Gazette

Production-ready alpha for **The Ozark Gazette** at `https://ozarks.cgnnews.net`.

Repo target: `CookInternational/Ozark-Gazette`.

## Contents

- CGN-style shell with Ozark Gazette header/footer and stock market ticker.
- Homepage with CGN LIVE-style layout based on the St. Louis affiliate pattern.
- Local, US, World, Politics, Investigations, Markets, Technology, Opinion, Environment, Entertainment and Obituaries category pages.
- Weather Center, Weather Radar, Traffic Center, Sports Center and Market Center.
- Horoscopes, Sudoku, Puzzles, Crosswords.
- Reporters page with Michael A. Cook as Editor and a Newsroom Reporter card.
- `apps-script/OzarkGazette.gs` for Ozark Gazette sheet routing and archival automation.

## Apps Script

Use the existing CGN Apps Script project/web app URL, and add `apps-script/OzarkGazette.gs` as a separate file. Add the router hook from `apps-script/OZARK_ROUTER_HOOK.txt` to the main route handler.
