OZARK GAZETTE ROUTER HOOK — REQUIRED IN THE SHARED CGN APPS SCRIPT

The Ozark Gazette now runs from the same CGN Apps Script web app URL as account/login/paywall logic:
https://script.google.com/macros/s/AKfycbx41mQg-Ine3XZ-VrMI_SaQn4_K6cDQHA0cBFyGPgupu_edNFoNRjSLv2hoSe_bOytt/exec

The file apps-script/OzarkGazette.gs is an extension file only. It does not add doGet/doPost.
Add this hook inside the existing CGN doGet(e), after payload/action normalization and before the default CGN fallback response:

  var ozarkResponse = ozarkGazetteRoute_(getRequestPayload_(e));
  if (ozarkResponse) return cgnJsonOrJsonp_(e, ozarkResponse);

If cgnJsonOrJsonp_ is not available in that script version, use the existing CGN json(...) helper:

  var ozarkResponse = ozarkGazetteRoute_(getRequestPayload_(e));
  if (ozarkResponse) return json(ozarkResponse);

Do not add another doGet or doPost to OzarkGazette.gs. CGN account/login/signup/paywall actions continue to run through the main CGN backend. Ozark article actions read only this spreadsheet:
1Xz9bnMqb-tkHeo2N2UonUbBr1jpo1VzKcVbBW_PU2n0

Required Ozark tabs: Articles, Archives, Obituaries.
